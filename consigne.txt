Recrutement front

Consigne :

L’exercice suivant a pour but d’avoir une vision globale de votre approche du code et de
l’utilisation de REACT JS. N’y passez donc pas trop de temps.
Grâce à ReactJs vous devez créer une petite app de ToDo liste s’approchant au mieux de
la maquette envoyée ( le pixel perfect n’est pas requis )

Attentes

Techniquement :
utilisation du système de composant de React
Utilisation d’un store ( redux ou context )
Utilisation des hooks

Les fonctionnalités de base attenduee :

ajouter des items à la to do liste
Voir les nouvelles tâches ajoutées
Pouvoir compléter des tâches
Voir les tâches complétées
Vous êtes libre pour le reste.

Rendu :

Votre code sous le format que vous souhaitez ( Git ou zip )
